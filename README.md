# Vellora — Public Pages

Static site hosting the Vellora privacy policy.

🔗 **Live**: https://vellora-app.github.io/

## Contents

- `index.html` — landing page with auto language detection
- `privacy.html` — Privacy Policy (Spanish)
- `privacy-en.html` — Privacy Policy (English)

## About Vellora

Vellora is an unofficial Emby client for Android and Android TV.

## Updating the privacy policy

1. Edit `privacy.html` and `privacy-en.html`
2. Update the "Last updated" date at the top of both files
3. Commit and push — GitHub Pages auto-deploys
